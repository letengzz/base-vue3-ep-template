# base-vue3-template

## 技术栈

- 开发框架：Vue3、TypeScript、Vite、Router、Pinia、axios、UnoCSS、Sass/Scss
- 代码规范：Eslint、Prettier、Stylelint、Husky、Lint-staged、cz-git

## 安装依赖

```sh
pnpm install
```

## 启动开发环境

```sh
pnpm dev
```

## 构建项目

```sh
pnpm build
```

## 运行测试

```sh
pnpm test:unit
```
